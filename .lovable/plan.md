

# Магическая книга «Таня Вайбкодер»

## Обзор
Интерфейс веб-приложения в виде открытой магической книги на космическом фоне. Два экрана: главный (ввод + результат) и каталог. Все элементы стилизованы под страницы книги — никаких стандартных UI-элементов.

## Структура

### 1. Космический фон и книга
- Тёмный фон с градиентами (фиолетовый/синий/розовый) и мягким свечением
- Изображение открытой книги (загруженное фото `forma_cl.png`) как основа интерфейса
- Книга центрирована, контент размещён поверх страниц

### 2. Левая страница — ввод
- Поле «Слово»: одна строка, крупный текст, тёмно-фиолетовый цвет, тонкая линия снизу, без рамок/фона
- Поле «Описание»: многострочное, рукописный шрифт, горизонтальные линии как в тетради, мягкое высветление зоны письма
- Плейсхолдер курсивный приглушённый
- Действия текстом: «сохранить | редактировать» — без кнопочного вида, свечение при наведении
- Ниже мелкий текст «Реакции»

### 3. Правая страница — результат
- Нумерованный список «1. слово — описание»
- Тёмно-синий/фиолетовый текст с лёгким свечением
- При пустом состоянии — подсказка

### 4. Плавающие слова вокруг книги
- 15 слов (VIBE, CODE, AI, FLOW, FOCUS, MAGIC, IDEA, SPEED, ПОТОК, ФОКУС, МАГИЯ, СМЫСЛ, НЕЙРО, ВАЙБ, КОД)
- Неоновое свечение, разные цвета, размеры
- Плавное движение вверх/по дуге, лёгкое вращение, «дыхание» opacity
- Разные длительности анимаций (20–40с)

### 5. Эффект переплёта
- Постоянное мягкое свечение в центре книги
- При сохранении — всплеск искр/частиц (золотой, розово-фиолетовый), вихрь вверх, затухание за ~1 сек

### 6. Анимации текста
- При вводе — появление с blur → чёткий
- На правой странице — эффект «написания» побуквенно или fade + сдвиг

### 7. Кастомный курсор
- SVG-перо или мягкое свечение точки вместо стандартного курсора

### 8. Экран каталога
- Список всех слов внутри страниц книги
- Нумерация
- Анимация перелистывания страниц (справа налево, тень, сгиб)

## Файлы
- `src/pages/Index.tsx` — основной экран с книгой
- `src/components/MagicBook.tsx` — компонент книги с левой/правой страницами
- `src/components/FloatingWords.tsx` — плавающие неоновые слова
- `src/components/SpineEffect.tsx` — свечение и частицы переплёта
- `src/components/CatalogView.tsx` — экран каталога с перелистыванием
- `src/index.css` — обновлённые стили, кастомный курсор, анимации
- Изображение книги скопировано в `src/assets/`
