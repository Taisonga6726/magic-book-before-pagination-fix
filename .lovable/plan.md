

# Словарный стиль: размеры, нумерация, превью шрифтов

## Проблемы
1. Слово слишком крупное (`text-3xl`) — нужно уменьшить до `text-xl` / `text-2xl`
2. Описание слишком мелкое (`text-sm`) — увеличить до `text-base`
3. Нет нумерации записей
4. Пользователь хочет увидеть превью разных каллиграфических шрифтов

## Изменения

### 1. Скорректировать размеры (`src/components/MagicBook.tsx`)
- Слово: `text-3xl` → `text-xl` (каллиграфический, Great Vibes)
- Описание: `text-sm` → `text-base` (Cormorant Garamond italic)
- Добавить нумерацию: перед каждой записью — номер `{i+1}.` мелким шрифтом

### 2. Превью шрифтов — временная секция
Добавить вверху правой страницы (или отдельным блоком) превью-сравнение 4 шрифтов на одном и том же слове «Магия»:
- **Great Vibes** — каллиграфический курсив
- **Dancing Script** — рукописный курсив  
- **Marck Script** — добавить импорт, русский рукописный
- **Cormorant Garamond italic** — книжный курсив

Каждый шрифт показан с подписью названия. Это временное превью для выбора — потом уберём.

### 3. Подключить Marck Script (`src/index.css`)
Добавить в Google Fonts import `Marck+Script` для русского каллиграфического варианта.

## Файлы
- `src/index.css` — добавить Marck Script в import
- `src/components/MagicBook.tsx` — размеры, нумерация, блок превью шрифтов
