## Контекст
Сейчас используется **Marck Script**. В проекте уже подключены 4 шрифта через Google Fonts:
- Marck Script
- Dancing Script
- Great Vibes
- Cormorant Garamond

Добавим ещё несколько популярных рукописных вариантов для сравнения.

## Изменения

### 1. `index.html` — подключить дополнительные шрифты
Добавить в `<head>` ссылку на Google Fonts с новыми шрифтами:
- **Pacifico** — округлый, игривый
- **Caveat** — лёгкий рукописный, хорошая читаемость
- **Comforter** — элегантный каллиграфический
- **Bad Script** — живой почерк, немного небрежный

### 2. Временный компонент `FontPreview.tsx`
Создать компонент-оверлей, который показывает текст «ВАйкобчанин» и «Tanya Gaiduk» каждым из 8 шрифтов в виде списка карточек. Каждая карточка:
- Название шрифта
- Превью текста крупно (как на странице книги)

### 3. `src/pages/Index.tsx`
Временно добавить `<FontPreview />` поверх книги для просмотра. После выбора шрифта — уберём.
