

# Перенести подпись "Tanya Gaiduk" на книгу

## Проблема
Подпись "Tanya Gaiduk" расположена в правом нижнем углу экрана. Стрелки на скриншоте указывают на нижнюю часть правой страницы книги — туда нужно переместить подпись.

## Изменения

### 1. `src/pages/Index.tsx` — убрать подпись из страницы
Удалить блок `div` с "Tanya Gaiduk" (строки 34-39).

### 2. `src/components/MagicBook.tsx` — добавить подпись на книгу
Внутри контейнера книги, добавить подпись в нижнюю правую часть правой страницы:
```tsx
<span
  className="absolute font-handwriting text-sm z-20 pointer-events-none select-none"
  style={{
    bottom: "15%",
    right: "15%",
    color: "hsl(40 80% 55% / 0.6)",
  }}
>
  Tanya Gaiduk
</span>
```

Подпись будет визуально находиться на правой странице книги, в нижней части — как авторская подпись на рукописи.
