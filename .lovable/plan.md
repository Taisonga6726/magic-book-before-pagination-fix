

## Диагноз

Сейчас структура записи (`renderEntry`) одинаковая для обеих страниц:
- Реакции выровнены `justify-end` (справа) — это правильно для левой страницы (направление к корешку), но НЕ зеркально для правой.
- Текст и описание — `text-align: left` / `justify` — одинаково для обеих.

По требованию пользователя:
- **Левая страница**: всё как сейчас, реакции справа (у корешка) — ✅ уже так.
- **Правая страница**: реакции должны быть **слева** (у корешка), зеркально левой.

То есть единственное реальное изменение — позиция реакций должна зависеть от того, на какой странице рендерится запись.
