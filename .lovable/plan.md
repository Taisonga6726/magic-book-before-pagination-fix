

# Убрать видимый квадрат баннера книги — варианты

## Проблема
Изображение `book.png` имеет прямоугольные границы, которые видны на фоне анимированного canvas — получается «квадрат» вокруг книги.

## Варианты решения

### Вариант A — CSS-маска с растушёвкой краёв (рекомендую)
Наложить на контейнер книги `mask-image: radial-gradient(...)` или `linear-gradient` по краям, чтобы границы изображения плавно растворялись в фон. Края book.png станут прозрачными, книга будет «выплывать» из космоса.

```css
mask-image: radial-gradient(ellipse 85% 90% at center, black 60%, transparent 100%);
```

### Вариант B — Подстроить цвета canvas под баннер
Изменить палитру в `HeroWave` (сдвинуть RGB-формулу), чтобы фон canvas был того же оттенка, что и края book.png. Тогда граница не будет видна, потому что цвета совпадут.

### Вариант C — Наложить gradient-overlay поверх краёв
Добавить 4 полупрозрачных градиента (сверху, снизу, слева, справа) поверх контейнера книги, которые затемняют края и сливают их с фоном. Похоже на виньетку.

## Рекомендация
**Вариант A + B вместе**: маска растворяет жёсткие края book.png, а цвета canvas подстраиваются под тон баннера. Результат — книга на сплошном космическом фоне без видимых границ.
