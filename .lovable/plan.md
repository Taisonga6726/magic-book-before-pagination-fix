## 2. Звук скрипа пера при печати
- Сгенерировать короткий аудиофайл звука пера программно (Web Audio API синтез) или использовать бесплатный MP3
- Добавить файл `public/pen-scratch.mp3` — короткий (~0.3-0.5 сек) звук скрипа пера
- В `MagicBook.tsx`: создать `Audio` объект, при каждом вводе символа в поле «Слово» проигрывать звук с рандомной позицией старта (`audio.currentTime = Math.random() * 0.3`)
- Звук воспроизводится тихо (`volume = 0.3`) чтобы не раздражать

### Техническая реализация звука
```ts
const penAudio = useRef(new Audio("/pen-scratch.mp3"));
penAudio.current.volume = 0.3;

// В onChange поля "Слово":
penAudio.current.currentTime = Math.random() * 0.3;
penAudio.current.play();
```
