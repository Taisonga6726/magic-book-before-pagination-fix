

# Эффект «запись пером» в реальном времени + toast «Запись внесена»

## Два изменения

### 1. Live-превью при печати на правой странице
Как только пользователь начинает печатать в поле «Слово», на правой странице появляется live-строка с чернильной анимацией — каждая новая буква проявляется с эффектом `ink-appear`.

**`src/components/InkWriteEffect.tsx`** — адаптация:
- Хранить предыдущий `text` через `useRef`
- Если новый `text` начинается со старого (пользователь дописывает) — `visibleCount` не сбрасывается, анимируются только новые символы
- Если текст изменился кардинально (удаление/замена) — сбросить и показать всё мгновенно

**`src/components/MagicBook.tsx`**:
- Убрать `writingIdx` — больше не нужен
- Пока `word` не пуст и запись не сохранена, на правой странице после всех `entries` показывать live-строку: номер `entries.length + 1` + `InkWriteEffect` с `text={word + (description ? ' — ' + description : '')}`
- При редактировании (`editIdx !== null`): live-строка заменяет запись по `editIdx`, а не добавляется в конец
- При `handleSave`: запись фиксируется в `entries` как обычный текст, live-превью исчезает (поля очищаются)

### 2. Toast «Запись внесена» при сохранении
**`src/components/MagicBook.tsx`**:
- Импорт `toast` из `sonner`
- В `handleSave` после сохранения: `toast("Запись внесена")`
