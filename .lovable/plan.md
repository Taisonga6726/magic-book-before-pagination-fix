

## Цель
Сильно усилить существующий `FinalBookMagicFX`: ярче центральное свечение, добавить ощущение света изнутри (по корешку), сияние страниц, больше и заметнее искр и дуг. Курсор-логику не трогаем.

## Файл: `src/components/FinalBookMagicFX.tsx` (правка только этого файла)

**A. Центральное свечение — ярче**
- Золото `rgba(255,200,80,0.55)` (было 0.28), фиолет `rgba(160,90,255,0.45)` (было 0.22)
- Доп. горячий core: белый `rgba(255,240,200,0.35)`, blur 30px
- Pulse 4s: opacity 0.85→1, scale 1→1.08

**B. Сияние вокруг страниц (новый слой)**
- Прямоугольник по форме разворота с двойным `box-shadow`:  
  `0 0 80px 20px rgba(255,200,100,0.35), 0 0 160px 40px rgba(160,90,255,0.25)`
- `mix-blend-mode: screen`, мягкая пульсация 5s в противофазе центру

**C. Свет изнутри (по корешку, новый слой)**
- Вертикальная полоса по центру разворота:  
  `linear-gradient(to right, transparent, rgba(255,220,140,0.45), transparent)`, ширина ~120px, blur 25px, screen blend
- Мерцание 3s (opacity 0.6→1)

**D. Дуги — заметнее**
- 3 эллипса (вместо 2), `stroke-width: 1.2–1.5`, `opacity: 0.6` контейнер + stroke 0.7
- `filter: drop-shadow(0 0 6px currentColor)` для свечения линий
- Вращение 25s / 35s / 45s в чередующихся направлениях

**E. Искры — больше и ярче**
- ~45 частиц (было 25), размер 1–3px
- Canvas `shadowBlur=8`, `shadowColor` = цвет частицы → glow
- Цвета: золото / фиолет / белый / тёплый янтарь
- Life 1.8–3.5s

## Что НЕ трогаем
- Курсор: halo + spawn по mousemove — без изменений
- `FinalBook.tsx`, текст, ink-эффект, layout, скрины, пагинация
- Другие экраны: `MagicBook`, `ControlBar`, `FloatingWords`, фон

## Критерии приёмки
- Книга — мощный источник энергии, заметно ярче нынешнего
- Свет ощущается идущим изнутри (корешок + центр + сияние страниц)
- Текст полностью читается (всё через screen blend)
- Стабильные ~60 FPS на 1240×861
- Эффекты только в FinalBook
