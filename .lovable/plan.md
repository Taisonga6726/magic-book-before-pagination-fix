

# Исправления по скриншоту + реалистичная SVG перьевая ручка

## Проблемы со скриншота

1. **Нумерация** — слишком мелкая и бледная, нужно увеличить и сделать ярче
2. **Описание** — слишком бледное (`opacity 0.7`), увеличить до `0.85`
3. **Текст выходит за орнамент** — правая страница: уменьшить `width` с `35%` до `30%`, добавить `paddingRight` побольше чтобы текст не залезал на рамку

## Изменения

### 1. `src/components/MagicBook.tsx` — layout и читаемость

- **Нумерация**: `text-sm font-semibold opacity-0.65` → `text-base font-bold` с `color: hsl(var(--ink) / 0.8)` — крупнее и чётче
- **Описание**: `color: hsl(var(--ink) / 0.7)` → `hsl(var(--ink) / 0.85)` — ярче
- **Правая страница**: уменьшить ширину контента чтобы не заходил за орнамент рамки. `width: "35%"` → `"32%"`, `padding-right: 28px`
- Аналогичные изменения для live preview строки

### 2. `src/components/QuillPen.tsx` — реалистичная перьевая ручка

Полная замена SVG — вместо птичьего пера рисуем перьевую ручку по мотивам фото:

- **Золотой nib (перо)**: расщеплённый кончик с градиентом `#c9a227` → `#8b6914`, тонкая гравировка-линия по центру
- **Чёрный корпус**: тёмный цилиндр с лёгким блеском (градиент `#1a1a1a` → `#333`)
- **Золотое кольцо**: декоративная полоска на стыке пера и корпуса
- **Капли чернил**: сохранить анимированные `ink-drop-1`, `ink-drop-2`
- **Размер**: ~24×44px, наклон 25-30° для естественного положения при письме
- Сохранить CSS-класс `quill-pen` и анимацию `quill-write`
