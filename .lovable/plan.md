

# Корректировка позиционирования контента на страницах книги

## Проблема (по скриншоту)
1. Контент на левой и правой страницах выходит за декоративные рамки книги
2. Нужно точно вписать контент внутрь орнаментальных границ каждой страницы
3. Космический фон должен покрывать всю страницу

## Изменения в `src/components/MagicBook.tsx`

### Левая страница (строка 70)
- Сдвинуть правее и ниже внутрь орнамента: `left: 8%` → `left: 10%`, `top: 12%` → `top: 15%`
- Уменьшить ширину и высоту чтобы не выходить за рамку: `width: 35%` → `width: 32%`, `height: 72%` → `height: 65%`
- Увеличить padding сверху и по бокам

### Правая страница (строка 101)
- Аналогично сдвинуть внутрь: `right: 8%` → `right: 10%`, `top: 12%` → `top: 15%`
- Уменьшить размеры: `width: 35%` → `width: 32%`, `height: 72%` → `height: 65%`

### Кнопка «каталог →» (строка 120)
- Поднять выше и сдвинуть внутрь рамки: `bottom: 6%` → `bottom: 12%`, `right: 12%` → `right: 14%`

### Контейнер книги (строка 57)
- Увеличить максимальную ширину для лучшего масштаба: `max-w-[960px]` → `max-w-[1100px]`

## Изменения в `src/pages/Index.tsx`
- Убедиться что `cosmic-bg` покрывает весь viewport: добавить `w-full h-screen` вместо `min-h-screen`
