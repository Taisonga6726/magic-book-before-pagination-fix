

# Сдвиг контента левой страницы правее

## Проблема
Поле «Слово» и текстовая область начинаются слишком левее — перекрывают орнамент. Нужно сдвинуть контент правее, чтобы текст начинался сразу после декоративного уголка.

## Изменения в `src/components/MagicBook.tsx`

### Левая страница (строка ~70)
- Сдвинуть правее: `left: 12%` → `left: 15%`
- Уменьшить ширину соответственно: `width: 28%` → `width: 25%`
- Увеличить левый padding: `28px` → `32px`
