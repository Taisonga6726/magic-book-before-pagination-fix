

# Исправить радужную анимацию кнопок

## Проблема
1. В градиенте есть зелёные цвета (`#1098ad`, `#07b39b`, `#6fba82`)
2. Анимация слишком быстрая (8s)
3. Слово «сохранить» не помещается в кнопку — недостаточный padding

## Изменения — 1 файл: `src/index.css`

### 1. Убрать зелёные цвета из градиента (строка 151)
Заменить 8-цветный градиент на 5-цветный без зелёного:
```css
background: linear-gradient(60deg, #f79533, #f37055, #ef4e7b, #a166ab, #5073b8);
```

### 2. Замедлить анимацию (строка 153)
`8s` → `16s`

### 3. Увеличить padding кнопки (строка 131)
`padding: 5px 18px` → `padding: 6px 24px`
