

## Диагноз

Сейчас на левой странице номер сидит в фикс-колонке `width: 2.4em`, а слово начинается в отдельной flex-колонке справа. Между «11.» и словом получается визуальный зазор ≈1.5–2 символа — это и есть «лишний пробел».

## Решение

В `src/components/FinalBook.tsx`, ветка `side === "left"` в `renderEntry`:

1. Убрать фикс-ширину у колонки номера. Вместо `flex` с двумя колонками сделать так:
   - номер и слово — **в одной строке inline** («11. Слово»), одним `<div>`, без flex-разрыва;
   - переносы длинного слова идут по обычному word-wrap, без лесенки, потому что весь блок «номер+слово» — единая строка с естественным `text-indent: 0` и `padding-left: 0`.
2. Описание остаётся отдельным `<div>` ниже, во всю ширину контейнера (как сейчас).
3. Реакции остаются в своём блоке снизу справа.
4. Правую страницу (`side === "right"`) не трогаю вообще.
5. Ширина измерительного блока пагинации уже учитывает 26/31 — оставляю.

## Изменение (только левая ветка `renderEntry`)

```tsx
if (side === "left") {
  return (
    <div key={globalIdx} className="flex flex-col mb-0 w-full">
      <div
        className="pb-0.5 text-xl leading-tight font-bold w-full"
        style={{
          color: "#1a1440",
          fontFamily: "'Cormorant Garamond', serif",
          fontStyle: "italic",
          lineHeight: "1.15",
          padding: 0,
          margin: 0,
        }}
      >
        {globalIdx + 1}. {renderInkWord(entry.word)}
      </div>
      {entry.description && (
        <div /* ...как сейчас, описание во всю ширину... */>
          — {entry.description}
        </div>
      )}
      {/* блок реакций — как сейчас */}
    </div>
  );
}
```

## Что НЕ трогаю
- Правую страницу (`side === "right"`) — ноль изменений.
- Координаты страниц, `top`, `height`, ширины 31% / 26%.
- Шрифты, цвета, ink-эффект, нумерацию, реакции, пагинацию.

## Проверка
1. Открыть финальную книгу на 1240×861.
2. Подтвердить: на левой странице между «11.» и словом обычный одиночный пробел, лесенки нет, описание идёт во всю ширину ниже.
3. Правая страница выглядит как до правки.
