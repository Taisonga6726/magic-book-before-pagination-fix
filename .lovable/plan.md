

# Сдвинуть плавающее слово влево от подписи

## Изменение — `src/components/FloatingWords.tsx`

Слово с индексом 9 (строка 30) наезжает на подпись. Сдвинуть его значительно левее:

```tsx
// Было:
{ bottom: "12%", right: "12%" },
// Стало:
{ bottom: "12%", right: "28%" },
```

Одна строка, одно изменение — `right` с `12%` на `28%`.
