

# Живой космический фон на canvas вместо CSS-градиентов

## Суть
Заменить статичный CSS-фон `.cosmic-bg` на анимированный canvas-фон (из предоставленного кода `HeroWave`). Canvas будет рендериться на весь экран за книгой — баннер книги сольётся с живым космическим фоном, и «квадрат» исчезнет.

## Изменения

### 1. Создать `src/components/ui/dynamic-wave-canvas-background.tsx`
Скопировать предоставленный компонент `HeroWave`, адаптировав под TypeScript:
- Типизировать `canvasRef` как `useRef<HTMLCanvasElement>(null)`
- Добавить null-check для canvas/ctx
- Canvas рендерится `position: fixed`, `inset: 0`, `z-index: 0` — за всем контентом

### 2. Изменить `src/pages/Index.tsx`
- Убрать класс `cosmic-bg` с корневого `div`
- Убрать `<div className="cosmic-glow-layer" />` (больше не нужен)
- Добавить `<HeroWave />` первым дочерним элементом — он будет фиксированным фоном за всей сценой
- Корневой div оставить с `relative overflow-hidden` и тёмным базовым `bg` как fallback

### 3. `src/index.css` — убрать `.cosmic-bg` и `.cosmic-glow-layer`
Удалить строки 226-243 (классы `cosmic-bg` и `cosmic-glow-layer`) — они больше не используются.

## Результат
- Фон — живая анимация волн в фиолетово-синих тонах на весь экран
- Книга «плавает» поверх космоса, баннер не виден как отдельный квадрат
- Плавающие слова остаются на своём слое поверх фона

## Технические детали
- Компонент рендерит `<canvas>` с `position: fixed; inset: 0; z-index: 0`
- Использует `requestAnimationFrame` для анимации — ~60fps
- `SCALE = 2` снижает нагрузку (рисует в половинном разрешении)
- Cleanup при unmount убирает resize listener
- Никаких внешних зависимостей не требуется
